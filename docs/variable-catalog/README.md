# 変数名目録

BQNコードを読むための補助資料置き場です。
コードの変数名を変更するためではなく、短い変数名・略語の意味を推定して読む助けにします。

## ファイル名ルール

```text
NNN_<source-file>__<function-or-area>__L<start>-L<end>.md
```

例:

```text
001_core.bqn__GetTxUpd__L121-L147.md
```

ファイル名だけで、どのファイルのどの範囲まで目録化済みか分かるようにします。
次回は、このディレクトリのファイル一覧を見て、未作成の範囲を小さく追加してください。

## 作成済み

- `001_core.bqn__GetTxUpd__L121-L147.md` — `core.bqn` の `GetTxUpd` 周辺
