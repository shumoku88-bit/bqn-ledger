# Bookkeeping Matrix Study: Service Payable Recognition and Payment

本フォルダは、簿記行列研究の第2題として、保守サービス費用の発生、未払金の計上、その後の支払いを検証するpublic synthetic fixtureです。

ここで英語の `payable` は広義の支払債務を指します。日本語の勘定分類では、通常の営業仕入ではない保守サービスの後払いを「買掛金」ではなく「未払金」として扱います。

## 1. 研究対象

- サービス費用の発生
- 未払金の計上
- 未払金の支払い
- 費用認識時点と現金支払い時点の分離

## 2. 手計算仕訳

### 取引1: サービス費用の発生と未払金計上（2026-09-15）

2,400円の保守サービスを受け、代金は後日支払う。

```text
借方  保守費用  2,400
貸方  未払金    2,400
```

### 取引2: 未払金の支払い（2026-09-25）

未払金2,400円を銀行口座から支払う。

```text
借方  未払金    2,400
貸方  銀行預金  2,400
```

## 3. 符号規約

Posting IRでは、借方方向を正、貸方方向を負とする。

- 費用の増加: 正
- 負債の増加: 負
- 負債の減少: 正
- 銀行資産の減少: 負

このfixtureでは未払金を `liabilities:other-payable` で表す。

## 4. 行方向の意味

`expected-event-account-matrix.tsv` の各event行は合計0になる。これは、取引ごとの借方合計と貸方合計が一致することを示す。

## 5. 列方向の意味

各account列をevent順に累積すると、各取引後の勘定残高が得られる。テストではBQNのscan `` +` `` を使い、期待するrunning balancesと完全一致させる。

## 6. 会計上の核心

費用は、保守サービスの提供を受けた時点で認識する。支払いが後日であることは、費用認識を支払日まで延期する理由にならない。

支払時には新しい費用を認識せず、未払金が消滅し、銀行預金が同額減少する。

## 7. 売掛金研究との鏡関係

- 売掛金研究: 収益認識と代金回収を分離した
- 本研究: 費用認識と代金支払いを分離する

fixtureとテストは相互依存させない。
